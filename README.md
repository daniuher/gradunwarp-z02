# gradunwarp fork

This gradunwarp is a fork of the og gradunwarp from WU - [github link](https://github.com/Washington-University/gradunwarp)).
This version does a small rewrite of the GradUnwarpRunner to allow the input of already evaluated spherical harmonics. 
As a result, this allows sharing the pre-evaluated file without distributing the proprietary .grad  

For all documentation concerning `gradunwarp`, please visit the original github repo. A
